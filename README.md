# Joint-Dual-Branch-Denoising-for-Underwater-Stereo-Depth-Estimation
This repository stores the denoising methods used in the paper “Joint Dual-Branch Denoising for Underwater Stereo Depth Estimation.”
Arrange the three modules as shown in the figure.
After integrating the three modules, end-to-end training and fine-tuning on datasets such as UWStereo and SQUID are critical.
